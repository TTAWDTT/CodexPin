This directory is reserved for the LXGW WenKai font files used by CodexPin.

To enable the custom font locally:

1. Download "LXGW WenKai" (Regular) from the official repository or a trusted mirror.
2. Place the `LXGWWenKai-Regular.ttf` file into this `assets/fonts` directory.
3. Restart the CodexPin Electron app.

If the font file is not present, CodexPin will fall back to system UI fonts and remain fully functional.
